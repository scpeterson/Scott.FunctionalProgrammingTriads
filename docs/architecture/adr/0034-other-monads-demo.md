# ADR 0034: Other Monads Demo

- Status: Accepted
- Date: 2026-03-10

## Context

An advanced survey demo is needed to expose additional monad patterns beyond the main triads.

## Decision

Keep `other-monads` as a standalone advanced monad survey demo.

## Consequences

### Positive

- Expands breadth beyond core monad triads.

### Negative

- Broader scope can reduce depth per concept.

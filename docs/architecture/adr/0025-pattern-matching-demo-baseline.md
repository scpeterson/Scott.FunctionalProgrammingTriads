# ADR 0025: Pattern Matching Demo Baseline

- Status: Accepted
- Date: 2026-03-10

## Context

Pattern matching is a key stepping stone from imperative branching toward functional style.

## Decision

Keep `pattern-matching` as a standalone baseline demo focused on C# pattern matching primitives.

## Consequences

### Positive

- Improves readability and declarative branching examples for newcomers.

### Negative

- Some overlap with other branching-focused demos.

# ADR 0031: Schrodinger's Cat Demo

- Status: Accepted
- Date: 2026-03-10

## Context

A metaphor-based demo helps explain monadic state/uncertainty concepts to imperative learners.

## Decision

Keep `demo-monad` as a dedicated narrative teaching demo.

## Consequences

### Positive

- Improves conceptual understanding with memorable domain framing.

### Negative

- Less production-oriented than workflow-focused triads.

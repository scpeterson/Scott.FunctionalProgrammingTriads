# Documentation Tags

Use these tags consistently across demo metadata (`IDemo.Tags`) and ADR prose.

## Learning Flow Tags

- `beginner`: Introductory material for imperative-first audiences
- `comparison`: Side-by-side imperative/C#/LanguageExt equivalence
- `advanced`: Less introductory, assumes FP familiarity

## Authoring Guidance

1. Prefer 3-7 tags per demo.
2. Include `comparison` when a demo has imperative/functional contrast value.
3. Include `beginner` only when the narrative avoids advanced jargon.
4. Reuse existing tags before introducing new ones.
